kerror
void kerror(error_level errlvl, int en_int, char *msg,...)
Prints information about the kernel. Disables interrupts so it will not get interrupted, checks to see if the error level is &gt;= the minimum level, and if so, prints the current clock tick, then prints the error message. Then, if en_int != 0, it enables interrupts again.
